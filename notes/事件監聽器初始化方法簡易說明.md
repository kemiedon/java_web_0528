# 📚 事件監聽器初始化方法簡易說明

## 🎯 整體目的
這個方法的工作是：**找出頁面上所有「加入購物車」按鈕，然後告訴它們「被點擊時該做什麼」**。

---

## 🔍 逐行解析

### 第1行：方法定義
```javascript
initializeEventListeners() {
```
定義一個叫做 `initializeEventListeners` 的方法。「Initialize」意思是「初始化」，「EventListeners」意思是「事件監聽器」，合起來就是「初始化事件監聽器」。

### 第2行：選取所有按鈕
```javascript
const addToCartButtons = document.querySelectorAll('.add-to-cart-btn');
```

**用人話解釋**：
- `document.querySelectorAll()` = 在整個網頁裡搜尋所有符合條件的元素
- `'.add-to-cart-btn'` = 搜尋所有 **CSS 類別為 `add-to-cart-btn`** 的元素
- `const addToCartButtons` = 把搜尋結果存到一個變數裡

**比喻**：就像在一個教室裡找出所有穿著紅色衣服的同學。

### 第3行：逐一處理每個按鈕
```javascript
addToCartButtons.forEach(button => {
```

**用人話解釋**：
- `forEach()` = 「對每一個進行操作」
- `button =>` = 對於每一個按鈕（取名為 `button`）

**比喻**：教室裡找到10個穿紅衣服的同學，現在要逐一跟每一個同學說話。

### 第4行：綁定點擊事件
```javascript
button.addEventListener('click', (e) => this.handleAddToCart(e));
```

**用人話解釋**：
- `addEventListener()` = 「幫這個按鈕安裝一個監聽器」
- `'click'` = 監聽「被點擊」這個事件
- `(e) => this.handleAddToCart(e)` = 當被點擊時，執行 `handleAddToCart()` 方法

**比喻**：就像跟每位同學說「如果有人敲你的肩膀，你就去廣播室」。

---

## 💡 整體流程圖

```
🌐 網頁載入
    ↓
🔍 搜尋所有 add-to-cart-btn 按鈕
    ↓
找到按鈕清單（例如4個按鈕）
    ↓
逐一為每個按鈕安裝 click 監聽器
    ↓
用戶點擊按鈕時 → 自動觸發 handleAddToCart()
```

---

## 🎬 實際運作場景

假設你的網頁有這樣的 HTML：

```html
<button class="add-to-cart-btn">加入購物車</button>
<button class="add-to-cart-btn">加入購物車</button>
<button class="add-to-cart-btn">加入購物車</button>
```

執行 `initializeEventListeners()` 後：
1. ✅ 找到3個按鈕
2. ✅ 第1個按鈕被點擊 → 觸發 `handleAddToCart()`
3. ✅ 第2個按鈕被點擊 → 觸發 `handleAddToCart()`
4. ✅ 第3個按鈕被點擊 → 觸發 `handleAddToCart()`

---

## ⚡ 關鍵要點

| 概念 | 說明 |
|------|------|
| **querySelector** | 找第一個符合條件的元素 |
| **querySelectorAll** | 找**所有**符合條件的元素（這裡用的） |
| **addEventListener** | 為元素「註冊」一個事件處理器 |
| **click** | 監聽使用者的點擊動作 |
| **forEach** | 依序對陣列中的每個元素做某件事 |
